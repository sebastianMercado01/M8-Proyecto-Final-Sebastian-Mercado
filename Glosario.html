Glosario
Atributo:
Los atributos en HTML se pueden considerar como información adicional que se coloca siempre en la etiqueta de apertura, esto para añadir, complementar información o significado al elemento HTML al que estamos colocando.
Embed:
Dentro de este elemento podemos insertar contenido externo a la web, como flash, java, etc.
Head:
Este elemento, define la cabecera de nuestro documento HTML, donde podemos hacer referencias a todos los scripts, hojas de estilos, titulo del documento, datos para buscadores, etc.
HTML:
Es el tag que define el comienzo y fin de un documento.
img:
Permite la inserción de imágenes de múltiples formatos en la web, como jpg, gif, png, etc.
Link:
Este elemento permite relacionar el HTML de nuestra web con el css. También admite otros tipos de documento, pero lo más común, es enlazar las hojas de estilos con las plantillas HTML.
Meta:
Este tag siempre se encuentra como parte de la información que proporciona la cabecera HTML (tag head) nos proporciona la información necesaria para indexar nuestra página web con los buscadores y facilitar su localización.
p:
Este es un tag de tratamiento de texto que nos permite crear parrafos en HTML.
Source:
Es un tag complementario que se utiliza con los elementos video y audio cuando se pretende que nuestro fichero tenga múltiples réplicas que permitan una mayor disponibilidad del contenido multimedia a reproducir.
Body:
Este elemento define el cuerpo principal del documento HTML que estamos editando. En un proyecto en html, este vendría siendo el texto que podemos ver en una página web.
